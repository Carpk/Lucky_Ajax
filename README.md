### Lucky_Ajax

Random dice roller, makes ajax request to server for each roll.

## Getting Started

Download and run the following commands:

`bundle`

`rake db:create`

`rake 'db:migrate`

`shotgun`

open browser to [http://127.0.0.1:9393/](http://127.0.0.1:9393/)
